# Raccoon Hunt

## Overview

**Raccoon Hunt** is an endless runner game where players must survive a relentless chase by a giant raccoon. Navigate through a road filled with trash dumpsters, dodge obstacles, and jump over hazards to outrun the beast for as long as possible.

---

## Story

A man wanders upon a road full of trash dumpsters and finds himself in a dangerous situation when a giant raccoon shows up. It chases him relentlessly, and the man must run for his life. He needs to dodge obstacles, jump over trash cans, and sprint endlessly to survive.

**Can you outrun the beast?**

---

## Game Concept

Raccoon Hunt is designed as a fast-paced, reflex-based endless runner game. The goal is simple: survive as long as you can.

**Core Ideas:**

* Endless forward movement
* Increasing difficulty over time
* Obstacle dodging and precise timing
* Simple controls, high challenge

---

## How to Play

Use your keyboard to survive as long as possible:

* **← Left Arrow** or **A** – Move left
* **→ Right Arrow** or **D** – Move right
* **Space** – Jump

Avoid obstacles and **don’t get caught by the raccoon!**

## Development Stack

* **Framework:** React.js

---

## Setup Instructions

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:

   ```
   npm install
   ```
4. Start the development server:

   ```
   npm start
   ```
5. Open the browser at the provided local address to run the game
